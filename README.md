# express-mern-x

Create a telegram Bot simply easy way.
it will be helpfull to easy setup a telegram server.Lets try and enjoy
.
## Create a project

```javascript
npx create-tg-bot project_name
or
npx create-tg-bot
```

## Run development

```javascript
npm run dev
```
